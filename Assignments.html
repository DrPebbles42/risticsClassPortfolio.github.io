Week 6

Week 6: Choose the Best OS

This topic is quite controversial because there really is no "best" OS. There is only the best OS for the particular user. Nonetheless, this assignment taught us to find the pros and cons of Windows, MacOS, and Linux for 3 different types of companies. Find the word doc here.

Summary

Interior Design

The logical answer for an Interior Design company would definitely be Windows. I chose Windows because the industry standard for Interior Design is Windows so it natively supports many of the programs.

Social Media Advertising

The OS I chose for this was Mac OS. This is becasue Apple's ecosystem allows for the user to go between different tools.

Nonprofit Social Service

For this, I chose Linux because it is cost effective since it is free! There are many distributions that are Windows like for those that are not tech savvy. Using Linux also means that the nonprofit would only need to spend the money on the machines that the users are going to use.
