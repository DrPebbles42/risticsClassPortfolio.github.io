Week 5

Week 5: Exploring Computer Hardware

For this discussion, we were tasked with answering 2 out of 7 prompts.

Personal Devices

Hardware in portable personal devices such as laptops, smartphones, and wearables differ from a standard desktop computer. The biggest difference is, of course, the size of the hardware in the device. Laptop hardware is usually layered and is a great deal smaller than your typical desktop. Desktops will need (usually) a separate screen, keyboard, mouse, camera, and other i/o devices to make the computer usable. Laptops tend to come with these things prebuilt in which is a reason that a lot of people opt for a laptop nowadays (other than the portability). Smartphones take this a step further and now the capabilities of a computer can fit in your pocket. Smartphones do not need any external i/o devices to work properly and as expected out of the box. There is a built in speaker, camera, phone service, and (usually) a touch screen to navigate the environment. The boards are much smaller in smartphones as are all of the connectors and the components on the board.

Component Fuctions

RAM is one of the most important parts of the computer. It is a stick of memory that is inserted into slots on the motherboard. RAM is a common culprit for windows blue screen of death errors. Having enough RAM on your computer and ensuring that it the memory on your device is not almost full is key to making sure that your device does not run sluggishly or start giving errors. So, how does it work? First, the data and programs are held in the HDD/SSD, thumb drive, or some other non-volatile memory when the computer is electronically powered off. Then, when the computer is turned on a program is started, the PC will copy the program from the non-volatile storage to the RAM, then run it. This is a great example of why the practice of actually turning your computer off (fully powered off, not just sleep) is so important. RAM is volatile memory which means that it is wiped every time the PC is turned off. Not restarting your computer sometimes will cause your computer to run sluggishly because the RAM is so full.

Week 7

Week 7: Programming Languages

For this dicussion, we once again needed to answer 2 of 7 prompts about programming languages.

Fundamental Differences

Machine language is the 0s and 1s that have been discussed prior. This is the language that speaks directly to the computer in binary to give it the instructions needed for it to work the way the user expects. Machine language is very difficult for humans to read because it's just bits and bits of binary. Machine language is written in binary.

Assembly Language is the language that is slightly more human readable, but unless there's a list, time, or someone who has been studying it for a long time it's hard to understand still, especially with high level languages out there. Assembly language is written in mnemonics (also known as symbolic names) to specify machine operations. This was huge because it meant that people no longer had to work with the 0s and 1s. Assembly language is run through an "assembler" which converts the assembly language to machine language so the hardware then knows what to do.

High Level Language is the language that's now most popular to be written amongst programmers. Some common examples include C, C++, Python, C#, Visual Basic, and more. It is written in almost plain English following very specific syntax that is unique to each language. These languages are run through a compiler which translates the code into machine language so the computer knows what to do.

Learning Curve

The easiest type of language to learn for beginners is a high level language. The most popular language for beginners is Python because it's easy to read and write. It's a more forgiving language compared to others like C. Machine and Assembly languages are much harder to grasp with machine language being the hardest. Machine language is literally 0s and 1s. I still stay away from binary as much as I can.

Week 8: Exploring Excel Macros

Excel macros are used to automate repetitive or tedius tasks to increase efficiency. There were some pretty cool tools that my classmates found. I chose something helpful, yet small.

Sub HighlightDuplicateValues() Dim myRange As Range Dim myCell As Range Set myRange = Selection For Each myCell In myRange If WorksheetFunction.CountIf(myRange, myCell.Value) > 1 Then myCell.Interior.ColorIndex = 36 End If Next myCell End Sub

This macro highlights duplicate cells to check for redundancy.

My Post:

Definitions

Internet is the electronic communications network that connects computer networks and organizational computer facilities around the world as defined by Merriam-Webster Dictionary. According to Cambridge, the internet is defined as the large system of connected computers around the world that allows people to share information and communicate with each other. The Internet is like the roads that communications (like emails) follow to go to another device. The world wide web exists on the Internet. The World Wide Web is the webpages that we visit every day. According to the Oxford Dictionary, it is defined as an information system on the internet which allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another. The Internet plays a role in global connectivity because it is the network that people connect to to be able to access the World Wide Web. The Web brings everyone to a place where they can share documents on the internet. It is the most common used resource on the Internet.

Underlying Technologies

The Internet uses many protocols to keep everything running smoothly. They are sets of rules that allows devices to communicate with each other over the Internet. Every device that is connected to the internet has a unique IP address that helps data know where to go and where it is coming from. IP addresses, which is unique to the device, helps the Internet distinguish between routers, computers, and websites. The steps of an internet protocol are: dividing data into packets, addressing, routing the packets, reassemble the data, and handle missing packets. Internet Protocols are needed because of speed mismatches between sender and receiver to ensure that all data gets to the destination without loss. Some of the protocols the internet uses are: TCP/IP, SMTP, PPP, FTP, SFTP, HTTP, HTTPS, TELNET, POP3, IPv4, IPv6, ICMP, UDP, IMAP, SSH, and Gopher. The Web also uses protocols on top of the Internet protocols being used. Some examples of the protocols the Web uses are: DNS and HTTP(S). Web servers are an important part of browsing the web. When the link is entered by a browser and the browser needs the information from the Web server to be delivered, the browser requests the file by HTTP. The request is then received by the Web server and accepts the request, finds the information requested, then delivers the information. Specifically, the browser will obtain the IP address and the browser will be brought to the web server. When clicking hyperlinks, which bring you to a specific piece of information on the Web, the browser will be brought to the requested page unless it no longer exists which will result in an error message.

Global Impact

The global impact due to the Internet and Web is insane. Communication between people is almost all digital now. Face to face conversations are becoming less common. More people are texting or using other social medias to stay in touch with their friends and family rather than calling them or going to visit. Commerce has also never been more booming. If ta business has a website, it is almost guaranteed to do better than if they are just open. This is because more people from around the world could buy the product not just those around. This also means that advertisements are everywhere on the Web. People getting their educations are able to do so without sitting in a classroom anymore. Just look at the World Campus option here at Penn State. We are able to get our degree completely online without ever having to step foot on campus.
